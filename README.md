# Simple Counter App

A clean, modern counter application built with pure HTML, CSS, and JavaScript.

## Features

- **Increment/Decrement**: Simple + and - buttons to adjust the counter
- **Reset**: Reset the counter to zero
- **Visual Feedback**: Color changes based on value (green for positive, red for negative, blue for zero)
- **Keyboard Support**: 
  - `+` or `=` to increment
  - `-` or `_` to decrement  
  - `R` to reset
  - `0` to set to zero
- **Touch Support**: Swipe gestures on mobile devices
  - Swipe right → increment
  - Swipe left → decrement
  - Swipe down → reset
- **Statistics**: Tracks total operations and last action
- **Responsive Design**: Works on all screen sizes

## How to Use

1. Open `counter.html` in any web browser
2. Click the **+** button to increment the counter
3. Click the **-** button to decrement the counter  
4. Click **Reset Counter** to return to zero
5. Use keyboard shortcuts for faster control
